# M.A.C.E 프로젝트

# Modular Adaptation for Cooperative Education

### 1. MACE란?


- Modular Adaptation : 모듈식 적응
- - 학습 내용을 작은 단위로 분해하여 학습자가 학습해야 할 내용을 구체화
- - 학생은 각 내용에 대한 자신의 이해도를 확인하고 선택하여 학습
- - AI에게 소통하며 각 문제에 맞는 피드백과 형성평가를 거침
- - 이해가 잘 안될 경우 이해도가 좋은 학생에게 도움을 요청할 수 있음

- Cooperatve Education : 협동 학습
- - 충분히 숙달한 학생들은 각 학습 영역에서 친구들에게 적절한 피드백을 제공함
- - 도움을 받은 학생은, 자신이 참고한 내용 중 가장 도움이 된 학생의 답변을 채택(복수)
- - 최종적으로 이질적인 집단을 구성함.
- - 학습목표에 맞는 핵심 과제를 협력적으로 접근하며 해결
